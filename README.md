# LineSoul: The Forgotten Field

This repository contains a skeleton Unity project for a 2.5D action RPG inspired by Lineage. The project includes basic scripts and directory structure to get started.

## Setup
1. Install Unity 2021 or later.
2. Clone this repository and open the `LineSoul` project in Unity.
3. Import or create models, animations, and prefabs as needed.

## Project Structure
```
Assets/
    Scripts/
        Player/
        Monster/
        Combat/
        System/
        UI/
```

## Running
Open the project in Unity, press `Play` to test. The sample scripts demonstrate simple movement, combat, monster AI, and item enhancement.
